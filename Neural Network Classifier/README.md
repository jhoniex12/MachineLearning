# Introduction

In this project built a classification model to distinguish the difference between Extra Virgin and Organic Extra Virgin olive oil samples using chemical composition features. The dataset is preprocessed and visualized using PCA and feature distribution analysis. A Gaussian Mixture Model is applied to generate synthetic training samples for the minority class to address class imbalance. The neural network classifier is evaluated using training accuracy, test accuracy, and classification reports to compare the effectiveness of Adam and SGD optimizers under different model configurations.

The two problems indicated above demonstrate how deep learning models can be applied to different types of machine learning tasks and how careful preprocessing, parameter tuning, and evaluation strategies contribute to reliable and meaningful model performance.
